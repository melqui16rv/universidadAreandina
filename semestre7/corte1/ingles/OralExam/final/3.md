Good morning everyone. Today, I am going to talk about a project called 'The Digital Vault'...

...which we recently implemented at my law firm.
**[ uich ui ri-cent-li im-ple-men-ted at mai lo-ferm. ]** **[PAUSA]**

The project aims to modernize our archives and protect sensitive information.
**[ De pro-yect eims tu mo-der-nais aur ar-caivs and pro-tect sen-si-tiv in-for-mei-shon. ]**

Before we started, the firm had struggled with disorganized paper records.
**[ Bi-for ui star-ted, de ferm jad stro-gold uid dis-or-ga-naist pei-per re-cords. ]**

We had lost two important cases because we couldn't find the evidence in time.
**[ Ui jad lost tu im-por-tant kei-ses bi-cos ui cud-ent faind de e-vi-dens in taim. ]**

Therefore, the partners decided to digitalize everything.
**[ Der-for, de part-ners di-sai-ded tu di-yi-ta-lais ev-ri-zing. ]** **[ASIENTE CON LA CABEZA]**

The process was intense. We started the pilot phase in July
**[ De pro-ces uas in-tens. Ui star-ted de pai-lot feis in yu-lai ]**

and purchased high-speed scanners in August.
**[ and pur-chast jai-spiid es-ca-ners in o-gost. ]**

Then, our team trained the junior lawyers in September.
**[ Den, aur tiim treind de yu-nior lo-iers in sep-tem-ber. ]**

So far, the results are outstanding. We have scanned over five thousand files.
**[ So far, de ri-solts ar aut-stan-ding. Ui jab es-cand o-ver faiv tau-sand fails. ]**

Although the transition was slow, our efficiency has improved by forty percent.
**[ Ol-dou de tran-si-shon uas slou, aur e-fi-shen-si jas im-pruvd bai for-ti per-sent. ]**

Furthermore, the team has implemented a new encrypted cloud system.
**[ Fur-der-mor, de tiim jas im-ple-men-ted a niu en-crip-ted claud sis-tem. ]**

Looking ahead, if we finish the database by December, the firm will host a celebration party.
**[ Lu-king a-jed, if ui fi-nish de dei-ta-beis bai di-sem-ber, de ferm uil jost a ce-le-brei-shon par-ti. ]**

However, we must be realistic. If we had more cloud storage right now, the backup would run much faster.
**[ Jau-e-ver, ui mast bi ria-lis-tic. If ui jad mor claud es-to-rash rait nau, de ba-cap ud ron mach fas-ter. ]**

Also, if we had started this project years ago, we would have saved a lot of paper.
**[ Ol-so, if ui jad star-ted dis pro-yect yiirs a-gou, ui ud jab seivd a lot ob pei-per. ]** **[MIRA FIJAMENTE A LA CÁMARA]**
