"[SONRÍE Y MIRA A LA CÁMARA] Gud mor-nin ev-ri-uan. Tu-dey, ai am go-ing tu tok a-baut a pro-yect cold 'De Di-yi-tal Volt', uich ui ri-cent-li im-ple-men-ted at mai lo-ferm. [PAUSA]
Bi-for ui star-ted, de ferm jad stro-gold uid dis-or-ga-naist pei-per re-cords. Ui jad lost tu im-por-tant kei-ses bi-cos ui cud-ent faind de e-vi-dens in taim. Der-for, de part-ners di-sai-ded tu di-yi-ta-lais ev-ri-zing. [ASIENTE CON LA CABEZA]
Ui star-ted de pai-lot feis in yu-lai and pur-chast jai-spiid es-ca-ners in o-gost. Den, aur tiim treind de yu-nior lo-iers in sep-tem-ber. So far, de ri-solts ar aut-stan-ding. Ui jab es-cand o-ver faiv tau-sand fails. Ol-dou de tran-si-shon uas slou, aur e-fi-shen-si jas im-pruvd bai for-ti per-sent. Fur-der-mor, de tiim jas im-ple-men-ted a niu en-crip-ted claud sis-tem.
Lu-king a-jed, if ui fi-nish de dei-ta-beis bai di-sem-ber, de ferm uil jost a ce-le-brei-shon par-ti. Jau-e-ver, ui mast bi ria-lis-tic. If ui jad mor claud es-to-rash rait nau, de ba-cap ud ron mach fas-ter. Ol-so, if ui jad star-ted dis pro-yect yiirs a-gou, ui ud jab seivd a lot ob pei-per. [MIRA FIJAMENTE A LA CÁMARA]
In con-clu-shon, dis pro-yect chein-yed jau ui uork. Tank iu for iur taim."
