# Texto en Inglés con su Pronunciación (Intercalada)
*Nota: Lee la parte en **negrita** para la pronunciación aproximada figurada al español.*

Good morning everyone. 
**[ Gud mor-nin ev-ri-uan ]**
Today, I am going to talk about a very important work project 
**[ Tu-dey, ai am go-ing tu tok a-baut a ve-ri im-por-tant uork pro-yect ]**
we recently carried out at my company. 
**[ ui ri-cent-li ca-rrid aut at mai com-pa-ni ]**
The project aims to improve delivery times 
**[ De pro-yect eims tu im-prub de-li-ve-ri taims ]**
and increase customer satisfaction.
**[ and in-cris cas-to-mer sa-tis-fak-shon ]**

To give you some context, 
**[ Tu guiv iu som con-text ]**
before we started this project, 
**[ bi-for ui star-ted dis pro-yect ]**
the company had struggled with constant shipping delays. 
**[ de com-pa-ni jad stro-gold uid cons-tant shi-ping di-leis ]**
We had lost several important contracts 
**[ Ui jad lost se-ve-ral im-por-tant con-tracts ]**
because our inventory system was completely obsolete. 
**[ bi-cos aur in-ven-to-ri sis-tem uas com-plit-li ob-so-lit ]**
Therefore, the management decided 
**[ Der-for, de ma-nash-ment di-sai-ded ]**
that we needed an urgent update.
**[ dat ui ni-ded an ur-yent ap-deit ]**

The implementation process was challenging but rewarding. 
**[ De im-ple-men-tei-shon pro-ses uas cha-len-ying bat ri-uar-ding ]**
We started the research phase last year. 
**[ Ui star-ted de ri-serch feis last yiir ]**
Then, we trained the employees in January, 
**[ Den, ui treind de em-ploi-iis in ya-nua-ri ]**
and our team finally launched the new platform in March.
**[ and aur tiim fai-na-li lonch-t de niu plat-form in march ]**

So far, the results are excellent. 
**[ So far, de ri-solts ar ek-se-lent ]**
We have reduced operational costs by twenty percent. 
**[ Ui jab ri-dust o-pe-rei-sho-nal costs bai tuen-ti per-sent ]**
Furthermore, the development team has implemented key changes 
**[ Fur-der-mor, de de-ve-lop-ment tiim jas im-ple-men-ted kii chein-yes ]**
in the warehouse interface. 
**[ in de uer-jaus in-ter-feis ]**
Although we faced some initial bugs, 
**[ Ol-dou ui feis-t som i-ni-shal bogs ]**
we have fixed them very quickly.
**[ ui jab fix-t dem ve-ri kuick-li ]**

Looking towards the future, we have great expectations. 
**[ Lu-king to-uards de fiu-chur, ui jab greit eks-pec-tei-shons ]**
If we keep updating the software regularly, 
**[ If ui kip ap-dei-ting de sof-uer re-gu-lar-li ]**
the project will save us a lot of money next year. 
**[ de pro-yect uil seiv as a lot ob mo-ni next yiir ]**
If we meet the final deadline next month, 
**[ If ui miit de fai-nal ded-lain next mond ]**
the company will give a bonus to all the staff members.
**[ de com-pa-ni uil guiv a bo-nus tu ol de staf mem-bers ]**

However, it is also important to reflect on the process. 
**[ Jau-e-ver, it is ol-so im-por-tant tu ri-flect on de pro-ses ]**
For example, if we had more engineers on the team right now, 
**[ For ek-sam-pol, if ui jad mor en-yi-neers on de tiim rait nau ]**
we would finish the mobile version much faster. 
**[ ui ud fi-nish de mo-bail ver-shon mach fas-ter ]**
Also, if we had started earlier, 
**[ Ol-so, if ui jad star-ted er-li-er ]**
we would have finished by now.
**[ ui ud jab fi-nish-t bai nau ]**

In conclusion, this project taught us 
**[ In con-clu-shon, dis pro-yect tot as ]**
the value of innovation and teamwork. 
**[ de va-liu ob i-no-vei-shon and tiim-uork ]**
Thank you very much for your attention.
**[ Tank iu ve-ri mach for iur a-ten-shon ]**
